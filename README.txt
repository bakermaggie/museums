PURPOSE
The purpose of this project was self-exploration with SQL, looking primarily at the finances of US museums. Questions that could be answered by this analysis are:
1. Which states have the highest average museum revenue? 
2. Which states have the highest average museum expenses?
3. Which states have the highest expense ratio?
4. Which states have the most museums per capita?

DATA
museums.csv
The museum dataset is an evolving list of museums and related organizations in the United States. The museum data was compiled from Institute of Museum and Library Services administrative records for discretionary grant recipients, IRS records for tax-exempt organizations, and private foundation grant recipients. Data accessed through Kaggle.
https://www.kaggle.com/datasets/imls/museum-directory?fbclid=IwAR0UIgal6SRO0kiYSf4hVzifTC2vD3YVAUo5MqMmK3AhzMvNA8wboqCkz5o

State Populations.csv
The population dataset is an unofficial population count as of 2018 from World Population Review. Data accessed through Kaggle.
